# 项目简介
> 一个用JavaWeb编写的下载站，没有用框架，单纯jsp文件实现。

# 目前已实现功能
- 支持文件及目录查看，下载 
- 支持自定义站点名称，站点描述
- 支持设置禁止访问目录
- 首页文件列表支持过滤，目前有以下选项
    1. 通过拓展名过滤
    2. 通过文件名过滤
    3. 通过目录名过滤
- 支持直接通过直链下载文件
- 等。。。(敬请期待)
